ChangeBamHeader

USAGE: ChangeBamHeader [options]

Add extra PGs into bam header, or change SM, LB or DS tag in RG line in header and this only works with one read group in the input bam. . Version: 0.04

OptionDescription
INPUT=FileThe input SAM or BAM file. Required.
OUTPUT=FileThe ouput SAM or BAM file. Required.
PG=StringThe extra PG line with fields separated by semicolon for bam header including ID, PN, VN and CL etc, PP id will be reset. Each field including tag name and value separated by colon. This option may be specified 0 or more times.
SAMPLE=StringThe sample name in the only RG group. Default value: null.
LIBRARY=StringThe library name in the only RG group. Default value: null.
DESCRIPTION=StringThe description in the only RG group. Default value: null.
