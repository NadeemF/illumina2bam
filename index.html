Overview of Illumina2bam Picard command-line tools

The Picard command-line tools are packaged as executable jar files. They require Java 1.6. They can be invoked as follows:

java jvm-args -jar PicardCommand.jar OPTION1=value1 OPTION2=value2...

Most of the commands are designed to run in 2GB of JVM, so the JVM argument -Xmx2g is recommended.

Standard Options

The following options are relevant for most Picard programs:

OptionDescription
--helpDisplays options specific to this tool.
--stdhelpDisplays options specific to this tool AND options common to all Picard command line tools.
--versionDisplays program version.
TMP_DIR=FileThis option may be specified 0 or more times.
VERBOSITY=LogLevelControl verbosity of logging. Default value: INFO. This option can be set to 'null' to clear the default value. Possible values: {ERROR, WARNING, INFO, DEBUG}
QUIET=BooleanWhether to suppress job-summary info on System.err. Default value: false. This option can be set to 'null' to clear the default value. Possible values: {true, false}
VALIDATION_STRINGENCY=ValidationStringencyValidation stringency for all SAM files read by this program. Setting stringency to SILENT can improve performance when processing a BAM file in which variable-length data (read, qualities, tags) do not otherwise need to be decoded. Default value: STRICT. This option can be set to 'null' to clear the default value. Possible values: {STRICT, LENIENT, SILENT}
COMPRESSION_LEVEL=IntegerCompression level for all compressed files created (e.g. BAM and GELI). Default value: 5. This option can be set to 'null' to clear the default value.
MAX_RECORDS_IN_RAM=IntegerWhen writing SAM files that need to be sorted, this will specify the number of records stored in RAM before spilling to disk. Increasing this number reduces the number of file handles needed to sort a SAM file, and increases the amount of RAM needed. Default value: 500000. This option can be set to 'null' to clear the default value.
CREATE_INDEX=BooleanWhether to create a BAM index when writing a coordinate-sorted BAM file. Default value: false. This option can be set to 'null' to clear the default value. Possible values: {true, false}
CREATE_MD5_FILE=BooleanWhether to create an MD5 digest for any BAM files created. Default value: false. This option can be set to 'null' to clear the default value. Possible values: {true, false}

Illumina2bam

USAGE: Illumina2bam [options]

Convert Illumina BCL to BAM or SAM file. Version: 0.04

OptionDescription
RUN_FOLDER=FileIllumina runfolder directory including runParameters xml file under it, upwards two levels from Intensities directory if not given. Default value: null.
INTENSITY_DIR=FileIllumina intensities directory including config xml file, and clocs, locs or pos files under lane directory. Required.
BASECALLS_DIR=FileIllumina basecalls directory including config xml file, and filter files, bcl, maybe scl files under lane cycle directory, using BaseCalls directory under intensities if not given. Default value: null.
LANE=IntegerLane number. Required.
OUTPUT=FileOutput file name. Required.
GENERATE_SECONDARY_BASE_CALLS=BooleanIncluding second base call or not, default false. Default value: false. This option can be set to 'null' to clear the default value. Possible values: {true, false}
PF_FILTER=BooleanFilter cluster or not, default true. Default value: true. This option can be set to 'null' to clear the default value. Possible values: {true, false}
READ_GROUP_ID=StringID used to link RG header record with RG tag in SAM record, default 1. Default value: 1. This option can be set to 'null' to clear the default value.
SAMPLE_ALIAS=StringThe name of the sequenced sample, using library name if not given. Default value: null.
LIBRARY_NAME=StringThe name of the sequenced library, default unknown. Default value: unknown. This option can be set to 'null' to clear the default value.
STUDY_NAME=StringThe name of the study. Default value: null.
PLATFORM_UNIT=StringThe platform unit, using runfolder name plus lane number if not given. Default value: null.
RUN_START_DATE=Iso8601DateThe start date of the run, read from config file if not given. Default value: null.
SEQUENCING_CENTER=StringSequence center name, default SC for Sanger Center. Default value: SC. This option can be set to 'null' to clear the default value.
PLATFORM=StringThe name of the sequencing technology that produced the read, default ILLUMINA. Default value: ILLUMINA. This option can be set to 'null' to clear the default value.
FIRST_TILE=IntegerIf set, this is the first tile to be processed (for debugging). Note that tiles are not processed in numerical order. Default value: null.
TILE_LIMIT=IntegerIf set, process no more than this many tiles (for debugging). Default value: null.
BARCODE_SEQUENCE_TAG_NAME=StringTag name for barcode sequence. Default value: BC. This option can be set to 'null' to clear the default value.
BARCODE_QUALITY_TAG_NAME=StringTag name for barcode quality. Default value: QT. This option can be set to 'null' to clear the default value.
SECOND_BARCODE_SEQUENCE_TAG_NAME=StringTag name for second barcode sequence. Default value: null.
SECOND_BARCODE_QUALITY_TAG_NAME=StringTag name for second barcode quality. Default value: null.

BamQualityQuantisation

USAGE: BamQualityQuantisation [options]

Quantise the original Illumina canned qualities into reduced resolution scores using bins 1->9=6,10->19=15,20->24=22,25->29=27,30->34=33,35->39=37,40->59=41. Version: 0.04

OptionDescription
INPUT=FileThe input SAM or BAM file. Required.
OUTPUT=FileThe ouput SAM or BAM file. Required.
USE_OLD_QUALITY=BooleanUse original quality scores from OQ tag. Default value: false. This option can be set to 'null' to clear the default value. Possible values: {true, false}

BamTagStripper

USAGE: BamTagStripper [options]

Strip a list of tags in bam/sam record. By default, any tags containing lowercase letters will be stripped and other tags will be kept. A list of tags can be given to keep or strip. Version: 0.04

OptionDescription
INPUT=FileThe input SAM or BAM file to strip. Required.
OUTPUT=FileThe stripped ouput SAM or BAM file. Required.
TAG_TO_KEEP=StringA list of tag containing lowercase letters to keep. This option may be specified 0 or more times.
TAG_TO_STRIP=StringA list of tag only containing uppercase letters to strip. This option may be specified 0 or more times.

ChangeBamHeader

USAGE: ChangeBamHeader [options]

Add extra PGs into bam header, or change SM, LB or DS tag in RG line in header and this only works with one read group in the input bam. . Version: 0.04

OptionDescription
INPUT=FileThe input SAM or BAM file. Required.
OUTPUT=FileThe ouput SAM or BAM file. Required.
PG=StringThe extra PG line with fields separated by semicolon for bam header including ID, PN, VN and CL etc, PP id will be reset. Each field including tag name and value separated by colon. This option may be specified 0 or more times.
SAMPLE=StringThe sample name in the only RG group. Default value: null.
LIBRARY=StringThe library name in the only RG group. Default value: null.
DESCRIPTION=StringThe description in the only RG group. Default value: null.

AlignmentFilter

USAGE: AlignmentFilter [options]

Give a list of SAM/BAM files with the same set of records and in the same order but aligned with different references, split reads into different files according to alignments. You have option to put unaligned reads into one of output files or a separate file. Version: 0.04

OptionDescription
INPUT_ALIGNMENT=FileThe input SAM or BAM file with alignment. This option must be specified at least 1 times.
OUTPUT_ALIGNMENT=FileThe ouput SAM or BAM file. It should have the same number of files as the input. This option must be specified at least 1 times.
OUTPUT_UNALIGNED=FileThe ouput SAM or BAM file for unaligned. If not given, the unaligned reads will put into the last output file. Default value: null.
METRICS_FILE=FileMetrics file name Default value: null.

SplitBamByReadGroup

USAGE: SplitBamByReadGroup [options]

Split a BAM file into multiple BAM files based on ReadGroup. Headers are a copy of the original file, removing @RGs where IDs match with the other ReadGroup IDs. Version: 0.04

OptionDescription
INPUT=FileThe input SAM or BAM file with multiple read groups. Required.
OUTPUT_PREFIX=StringThe prefix for output bam/sam file. Required.
OUTPUT_COMMON_RG_HEAD_TO_TRIM=StringThe common RG name head to trim from the output file name. Default value: null.

BamMerger

USAGE: BamMerger [options]

A command-line tool to merge BAM/SAM alignment info in the first input file with the data in an unmapped BAM file, producing a third BAM file that has alignment data and all the additional data from the unmapped BAM. Version: 0.04

OptionDescription
ALIGNED_BAM=FileThe input SAM or BAM file with alignment. Required.
ALIGNMENT_PROGRAM_ID=StringThe alignment program ID in the header of the SAM or BAM file with alignment. Default value: bwa. This option can be set to 'null' to clear the default value.
KEEP_ALL_PG=Booleankeep all other PGs in aligned bam into output Default value: false. This option can be set to 'null' to clear the default value. Possible values: {true, false}
INPUT=FileThe input unmapped SAM or BAM file to merge. Required.
OUTPUT=FileThe output file after merging. Required.
KEEP_EXTRA_UNMAPPED_READS=BooleanKEEP extra unmapped reads in unmapped bam file to the final output if true. Default value: false. This option can be set to 'null' to clear the default value. Possible values: {true, false}
REPLACE_ALIGNED_BASE_QUALITY=BooleanReplace base qualities in aligned bam wtih the ones in unaligned bam if true. Default value: false. This option can be set to 'null' to clear the default value. Possible values: {true, false}
