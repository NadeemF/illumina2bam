BamTagStripper

USAGE: BamTagStripper [options]

Strip a list of tags in bam/sam record. By default, any tags containing lowercase letters will be stripped and other tags will be kept. A list of tags can be given to keep or strip. Version: 0.04

OptionDescription
INPUT=FileThe input SAM or BAM file to strip. Required.
OUTPUT=FileThe stripped ouput SAM or BAM file. Required.
TAG_TO_KEEP=StringA list of tag containing lowercase letters to keep. This option may be specified 0 or more times.
TAG_TO_STRIP=StringA list of tag only containing uppercase letters to strip. This option may be specified 0 or more times.
