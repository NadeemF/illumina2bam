BamReadTrimmer

USAGE: BamReadTrimmer [options]

Strip part of a read in fixed positionos, optionally place this and its quality in BAM tags. Version: 0.04

OptionDescription
INPUT=FileThe input SAM or BAM file to trim. Required.
OUTPUT=FileThe output file after trimming. Required.
ONLY_FORWARD_READ=BooleanJust trim the forward read if true. Default value: true. This option can be set to 'null' to clear the default value. Possible values: {true, false}
FIRST_POSITION_TO_TRIM=IntegerFirst position to be trimmed. Required.
TRIM_LENGTH=IntegerThe lenght to be trimmed. Required.
SAVE_TRIM=BooleanTimmed bases to be saved? Default value: true. This option can be set to 'null' to clear the default value. Possible values: {true, false}
TRIM_BASE_TAG=StringTag name to be used for timmed bases. Default value: rs. This option can be set to 'null' to clear the default value.
TRIM_QUALITY_TAG=StringTag name to be used for timmed qualities. Default value: qs. This option can be set to 'null' to clear the default value.
TAG_TO_KEEP=StringA list of tags to keep. This option may be specified 0 or more times.
